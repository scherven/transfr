Map data © OpenStreetMap contributors
Every station map, platform, footway, level and facility in Transfr is derived from OpenStreetMap, available under the Open Database License (ODbL). © OpenStreetMap contributors.
Data sources
OpenStreetMap
Base map, platforms, walkways, levels, facilities. © OpenStreetMap contributors, ODbL. openstreetmap.org/copyright
Transitous · MOTIS
Journeys, live delays and platform assignments, via the open Transitous routing service (MOTIS). Community open data.
Deutsche Bahn · IRIS
Real-time platform and track changes for German stations, via DB's public IRIS feed.
Coach-formation providers
Train coach order & sector maps from the operators' formation APIs (DB Wagenreihung and equivalents), where published.
Built with
Regions are extracted offline from the OSM planet with osmium; routing runs on the local core/ pathfinder. No map tiles are fetched at run time — an installed region is fully self-contained.
ODbL requires that any public use of this data keep the “© OpenStreetMap contributors” credit and share adaptations alike.
